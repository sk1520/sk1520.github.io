MTA Android App

In this project, my peer and I worked on an android app. Our finished app is a utility app for New York's transit system MTA, the app can do various things to help people who rely on the MTA. Completing this project taught me very important things, it taught me using Android Studios, testing apps, catching bugs and fixing them, designing UX/UI, using outside libraries, using company’s public data, and much more, but most importantly it taught me to work with someone else. Working with someone else, I learned that it's very different than working by yourself, in the process of working with my peer, I learned that the pacing is faster, and that communication is absolutely crucial in having a smooth process, me and my peer constantly communicated about our parts of assignments, we asked for help from each other when one got stuck, we asked for feedback, and constantly updated each other about our parts of assignments, being this is our first big project that we both worked on, it went very well without much roadblocks because we constantly communicated with each other.

    App Features:

  • Balance Calculator: An MTA fare is $2.75 and because the MTA gives a 5% bonus to your card with $5.50 purchase or more many riders always have balance left over on their cards and some even discard their cards with some type of balance left over because they can't get it down to $0. Our calculator calculates exactly how much you need to put in to your card so that when it's used it will always have a $0 balance.
  • MTA Service Status: Using MTA data we were able to implement a real-time service status checker for buses, trains, tunnels, bridges, and Long Island Rail Road.
  • Other features: Custom tab of MTA train and bus tracker, display of service prices, links to useful MTA information, maps of all transportation services, MTA contact information, user logs saved in an SQLite database, and a user submitted feedback system for bugs and suggestions.

Crypto Currency Ranker

In this project i created a simple non-bloated website that would rank top 100 crypto currencies and will keep track of price changes, and their position in the list. I took on this challenge to gain experience working with a company’s public JSON API data. Using AJAX technique i was able to retrieve JSON data from Coinmarketcap's public API and display it in a clean, aesthetic, and easy to read format.

    Site Features:

  • Live Updates: Crypto currency refreshes every 5 seconds ensuring up to date prices are being displayed.
  • Percent Changes: Percent changes are displayed and changes colors to highlight if negative or positive change in certain intervals.
  • Mobile Support: Supports mobile viewings, enlarges displayed information so mobile viewings will be able to read data.
